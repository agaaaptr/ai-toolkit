## Existing-docs decision (scan step)
If the project already has a doc structure (e.g. `vibes/`, `docs/superpowers/`, a custom layout) or similar docs → ASK the user:
- (a) **Adapt** the existing to the standard (move/rename), OR
- (b) **Leave** the existing as-is and generate the standard alongside.
Never overwrite or delete existing docs without explicit confirmation.
